# Duma Pure Technologies  
## About This Project

A fast, modern, fully responsive single-page website built with **pure HTML, CSS, and vanilla JavaScript** – no frameworks, no bloat.

Designed for **Duma Pure Technologies**, a South African tech company offering:
- Web Development
- Cybersecurity
- Laptop Repair & Services
- Graphic Design
- AI Solutions & Automation
- Digital Marketing
- Search Engine Optimization (SEO)

Perfect for startups, freelancers, or small agencies wanting a beautiful, high-performance online presence.

## Features

- Mobile-first responsive design (hamburger menu on mobile)
- Smooth scrolling & scroll-triggered animations
- Dark / Light mode toggle
- Live chat powered by tawk.to with Apollo AI assistant
- Real working contact form (Formspree)
- SEO-optimized & accessible
- Lightning-fast loading (no external frameworks)
- Beautiful navy blue + yellow branding
- Testimonials section
- Favicon & logo integration

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript (ES6+)
- Lucide Icons (lightweight SVG icons)
- Google Fonts – Poppins
- Hosted on GitHub Pages (free & instant)

## Customization

- Change colors in `:root` (CSS variables): `--primary` (navy), `--accent` (yellow)
- Add/remove services in the `#services` grid
- Update testimonials with real client quotes
- Replace placeholder images with your own
